# 📌 Project Overview

This project is a front-end clone of the main homepage of Amazon, built using only HTML and CSS. It recreates the layout, structure, and visual design of the original website while focusing on clean code and responsive design principles.

# The purpose of this project is to practice and strengthen foundational front-end development skills without using JavaScript.

# 🚀 Features

# Responsive navigation bar with logo and search bar design

# Product category sections

# Promotional banners

# Product grid layout

# Styled footer section

# Fully responsive layout using CSS

# 🛠️ Technologies Used

# HTML5

# CSS3 (Flexbox & Grid)

# 🎯 Purpose

# This project was created for educational purposes to improve understanding of webpage structure, layout design, and responsive styling using only HTML and CSS. It is not affiliated with or endorsed by Amazon.
